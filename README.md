# Transferência Backend Challenge

API REST desenvolvida em Java com Spring Boot para simular transferências financeiras.

## 🚀 Tecnologias

- Java
- Spring Boot
- PostgreSQL
- Docker

## 📌 Funcionalidades

- Transferência entre usuários
- Validação de saldo
- Integração com API externa

## ▶️ Como rodar

```bash
mvn spring-boot:run
